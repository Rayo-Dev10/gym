# Horario Gimnasio

Este repositorio muestra el horario del gimnasio del IES CINOC para el grupo B-2025. La página `index.html` incluye el estilo, los datos del horario en formato JSON y un pequeño script que genera la tabla automáticamente.

Para modificar el horario abre `index.html` y edita el bloque JSON identificado por `horario-data`. Al cargar la página en el navegador se actualizará la información.
