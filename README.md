# Horario Gimnasio

Este repositorio contiene el horario del gimnasio del IES CINOC para el grupo B-2025. La intención es mantener la información de manera sencilla en archivos separados.

- **index.html**: maquetación de la página principal.
- **styles.css**: hojas de estilo utilizadas por la página.
- **data.json**: datos del horario en formato JSON.

La tabla del horario actualmente está escrita de forma estática en `index.html`. Para que se cargue de forma dinámica desde `data.json` es necesario usar un motor de plantillas o algún script que procese el JSON antes de servir la página.
